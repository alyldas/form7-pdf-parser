# Support

Use GitHub Discussions for usage questions and GitHub Issues for reproducible bugs and feature
requests. Security problems must be reported privately as described in [SECURITY.md](SECURITY.md).

Before asking for help:

- confirm the PDF has an extractable text layer;
- reproduce the problem with the latest release;
- reduce it to a newly generated synthetic document;
- remove all personal, payment, and tracking data.

This volunteer project provides no service-level agreement or official carrier support.
